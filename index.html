node-dijkstra

JavaScript implementation of the Dijkstra shortest path problem

Quick Start

If you are already familiar with the Dijkstra's Shortest Path Problem and just want to get the feel of the library that's the right place to start. You then might want to check out the full API. 


const Graph = require('node-dijkstra')

const route = new Graph()

route.addNode('A', { B:1 })
route.addNode('B', { A:1, C:2, D: 4 })
route.addNode('C', { B:2, D:1 })
route.addNode('D', { C:1, B:4 })

route.path('A', 'D') // => [ 'A', 'B', 'C', 'D' ]
